System Specifications

Built for terminal enthusiasts who miss the golden age of text-based adventures.

▓▓▓

Terminal Native

Experience gaming the way the pioneers did. Pure ASCII and Unicode graphics render directly in your terminal with zero browser overhead.

$ cargo install fenrok
◈◇◈

Real-time Multiplayer

Adventure alongside other players in a persistent world. See characters move, chat, and battle in real-time through SpacetimeDB.

$ connect maincloud.spacetimedb.com
┌─┐

Roguelike Roots

Procedural dungeons, permadeath zones, and strategic turn-based combat. Every run tells a different story.

$ enter --dungeon abyss
†╬†

Classic MMO Soul

Character classes, skill trees, equipment crafting, guilds, and trading. All the depth you love, rendered in glorious Unicode.

$ char --class warrior
░▒▓

Cross-Platform

Linux, Windows, macOS—if it runs a terminal, it runs Fenrok. SSH into your server and play from anywhere.

$ ssh fenrok@myserver
